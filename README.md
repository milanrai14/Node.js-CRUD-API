
A simple RESTful API built with **Node.js**, **Express**, **MongoDB**, and **Multer** for handling **blogs** with support for **multiple image uploads**.

---

## Features

- Create a blog with **multiple images**
- Fetch all blogs or a single blog
- Update a blog and replace images
- Delete a blog along with its images
- Built using **Express**, **MongoDB (Mongoose)**, and **Multer** for file uploads
