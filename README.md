# Node.js CRUD Blog API | Express & MongoDB Atlas

A RESTful API built with **Node.js**, **Express**, and **MongoDB Atlas** that allows users to perform full CRUD operations on blog posts. The API supports **multiple image uploads** using Multer, stores files on the server, and saves metadata in MongoDB.
